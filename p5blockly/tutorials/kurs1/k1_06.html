K1.6a: Kopien von Figuren

Gleiche Kreise

  • Im letzten Kapitel wurden vier gleiche Kreise gezeichnet.

  • Dafür wurden 4 Blöcke eingebaut.

  • Solche Wiederholungen kann man mit einem Wiederholungs-Block zusammenfassen. Da sich die x-Koordinate der 4 Kreise ändert, brauchen wir eine neue Variable für die x-Koordinate. Der Wert der Variable "xkoord" wird bei jedem Durchlauf durch die Wiederholungsschleife geändert.

 K1.6b: Wiederholung einer Wiederholung

Mehrere Reihen von Kreisen

  • Wenn man einen Wiederholungs-Block in einen Wiederholungsblock einbaut, dann können mehrere Reihen von Kreisen gezeichnet werden. Dazu brauchen wir eine weitere Variable für die y-Koordinate.

  • Der äußere Wiederholungsblock wird 4 mal wiederholt. Der innere Wiederholungsblock wird 4 x 4 = 16 Mal wiederholt. Damit die Kreise immer wieder vom Anfang der Zeile aus gezeichnet werden, muss vor jedem Neustart des inneren Wiederholungsblocks die x-Koordinate auf 50 zurückgesetzt werden.

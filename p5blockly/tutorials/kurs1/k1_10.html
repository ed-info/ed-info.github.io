K1.10a: Text anzeigen

Linksbündig, rechtsbündig, zentriert

  • Text wird an der angegebenen Position angezeigt. Die Textgröße kann eingestellt werden und man kann den Text linksbündig, rechtsbündig und zentriert anzeigen lassen. Mit der Liniendicke, Linienfarbe und Füllfarbe kann das Aussehen des Textes bestimmt werden.

 K1.10b: Textbox anzeigen

Viel Text in der Box

  • Wenn man viel Text anzeigen möchte, dann kann man das mit einer Textbox machen, die sich um die Zeilenumbrüche automatisch kümmert.

 K1.10c: Text skalieren

Textgröße ändern

  • Die Textgröße kann Pixelgenau eingestellt werden, so dass auch sehr großer Text möglich ist.

 K1.10d: Text bewegen

Der Text folgt der Maus

  • Text kann wie andere Figuren mit Hilfe der Maus über die Zeichenfläche bewegt werden.

 K1.10e: Text rotieren

Text kann gedreht werden

  • Mit Hilfe der Blöcke "Verschieben" und "Drehung" kann ein Text verschoben und gedreht werden.

 K1.10f: Text vergrößern

Text kann vergrößert werden

  • Auch die Vergrößerung von Text funktioniert genauso wie bei den anderen Figuren.

 K1.10g: Text zusammensetzen

Text aus Variablen und Zeichenfolgen zusammensetzen

  • Aus einzelnen Textstücken und Variablen kann neuer Text angezeigt werden. Auf diese Weise kann auch der Inhalt von Texten mit Hilfe von Variablen verändert werden.

 K1.10h: Text eingeben

Text eingeben und in einer Variablen speichern

  • Man kann ein oder mehrere Texteingabefelder in die Zeichenfläche einfügen. Dort kann Text eingetippt werden, der in einer Variablen gespeichert wird. Der gespeicherte Text kann wieder ausgegeben werden.

  • Jedes Textfeld braucht einen eigenen Namen.

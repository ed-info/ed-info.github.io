S1.6 - Spiralen

Vorlage laden:
Eine Spirale

  • Die Schildkröte soll eine Spirale zeichnen. Eine Spirale sieht z.B. wie folgt aus:

    Wenn man die Spirale betrachtet, fällt auf, dass die Strecke, welche die Schildkröte vorwärts geht, nach jeder Richtungsänderung länger wird. Der Richtungswinkel ist 45° groß. Das soll der Schildkröte jetzt beigebracht werden.

    Im Programm wird eine Variable "streckenlaenge" festgelegt, welche nach jeder Richtungsänderung erhöht wird. Dazu klickt man in der Kategorie "Variablen" auf "Variable erstellen..." und nennt die neue Variable z.B. "streckenlaenge".

    In das Programm wird der Block "setze streckenlaenge auf" eingefügt und mit dem Wert 0 ergänzt

    Die Wiederholungsschleife wird 150 mal wiederholt und der Drehwinkel wird auf 45° gesetzt.

Richtungswinkel gleich 120°

  • Bei einem Richtungswinkel von 120° ist die Spirale dreiecksförmig. Hier kann man auch die Änderung der Streckenlänge verändern. Auch die Änderung der Liniendicke verändert die entstehende Spirale. Einfach ausprobieren.

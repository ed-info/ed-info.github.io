S1.4 - Variable Seitenlänge

Vorlage laden:
Seitenlänge des Quadrats ändern

  • Wenn die Strecke, welche die Schildkröte vorwärts geht, verkürzt oder verlängert wird, entstehen unterschiedlich große Quadrate.

    Für jedes Quadrat legt man die Seitenlänge fest und erstellt eine Wiederholungsschleife, um das Quadrat zu zeichnen.

Beispielprogramm laden:
Seitenlänge mit einer Variablen verändern

  • Eine Größe, die sich während eines Programmablaufs ändert, nennt man eine Variable. Die Seitenlänge des Quadrats soll mit Hilfe einer Variablen verändert werden.

    Um eine Variable festzulegen, klickt man in der Werkzeugleiste auf "Variablen". Es sind bereits die Variablen vorhanden, die für die Zeichenfläche und die Schildkröte benötigt werden. Diese sollten nicht verändert werden, damit das Programm fehlerfrei ablaufen kann.

    Um eine neue Variable festzulegen klickt man auf "Variable erstellen..." und gibt als Variablennamen z.B. den Namen "seitenlaenge" ein. Im Variablennamen sollten keine Umlaute oder Sonderzeichen verwendet werden.

    Die neue Variable "seitenlaenge" wird in die Liste der vorhandenen Variablen eingefügt.

    Die Variable "seitenlaenge" wird in das Programm eingefügt, indem man den Block "setze xxxxxx auf" in das Programm einfügt, Die "xxxxxx" stehen für irgendeinen Variablennamen.

    Falls als Variablenname nicht bereits "seitenlaenge" angezeigt wird, klickt man auf den Variablennamen und klickt in der Liste der Variabeln die Variable "seitenlaenge" an.

    Als Wert der neuen Variablen soll die Zahl 10 festgelegt werden. Dazu fügt man aus der Kategorie "Mathematik" den Zahlblock an die Variable an und setzt den Zahlenwert auf 10.

    Schließlich ersetzt man im Block "Vorwärts" den Zahlenwert mit dem Variablennamen, indem man den Block "seitenlaenge" aus der Kategorie "Variablen" holt und den Zahlenwert im Block "Vorwärts" damit ersetzt.

    Die beiden anderen Wiederholungsblöcke brauchen wir nicht mehr. Diese können gelöscht werden. Dazu zieht man den Block "Bewegung aufzeichnen" unter den ersten Wiederholungsblock und schiebt die anderen beiden Wiederholungsblocks auf den Mülleimer.

    Das Programm sollte jetzt wie folgt aussehen. Die Seitenlänge kann man ändern, indem man den Zahlenwert im Block "setze seitenlaenge auf" ändert.

Beispielprogramm laden:
Viele verschiedene Quadrate zeichnen

  • Mit Hilfe der Variablen "seitenlaenge" können viele Quadrate mit größer werdender Seitenlängen automatisiert gezeichnet werden.

    Dazu fügt man eine zweite Wiederholungsschleife in das Programm ein, die dafür sorgt, dass bei jedem Durchlauf die Variable "seitenlaenge" um einen bestimmten Betrag vergrößert wird.

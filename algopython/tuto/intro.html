Les niveaux qui vont suivre ont pour but l'apprentissage de la programmation en langage informatique Python.

Les applications que nous utilisons sur nos appareils informatiques (ordinateurs, téléphones, tablettes...) ont été créées grâce à plusieurs langages de programmation.
Il existe beaucoup de langages : le C, un des plus anciens, le Java, très utilisé sur de nombreux appareils, le JS qui fonctionne dans les navigateurs web... D'autres langages comme le HTML ou le CSS permettent la conception de sites Internet. Tous ces langages sont « compris » par les ordinateurs pour afficher et faire fonctionner les applications.

Et le Python dans tout ça ? C'est un langage récent, de plus en plus répandu, et aussi un des plus simples, avec peu de mots clés à retenir ou de symboles à utiliser. Ainsi, le Python est le langage idéal pour s'initier à l'algorithmique et à la programmation.

Dans AlgoPython, vous apprendrez le Python en relevant des défis de plus en plus complexes.
Chaque défi vous demandera d'écrire un programme en Python pour afficher des messages, reproduire un dessin, déplacer un personnage, effectuer des calculs...
Voici comment cela se présentera :
  • À gauche se trouve l'espace dans lequel vous taperez du code Python.
  • Pour tester si ce code fonctionne bien, il faut cliquer sur le bouton vert « Exécuter ».
  • Le résultat s'affichera dans l'espace à droite !
Il y aurait encore beaucoup à dire sur AlgoPython ou sur le langage Python, mais le mieux est de tester par vous-même !
Cliquez sur « Continuer » pour relever votre premier défi en Python !

#Атрибути вводу

Тег input є порожнім елементом, що ідентифікує конкретний тип інформації поля для отримання від користувача.

<input type="text" name="?" value="?" minlength="6"	 required />
-
type="…" Тип даних, що вводяться
value="…" Значення за замовчуванням
name="…" Використовується для опису цих даних у HTTP-запиті
id="…" Унікальний ідентифікатор, який використовують інші HTML-елементи
readonly Запобігає модифікації користувачем
disabled Запобігає будь-якій взаємодії
checked Чи вибрано радіокнопку або прапорець
required Обов'язковість заповнення, див. required
placeholder="…" Додає тимчасовий текст, див. ::placeholder
autocomplete="off" Вимкнути автозаповнення
autocapitalize="none" Вимкнути автоматичну капіталізацію
inputmode="…" Відобразити конкретну клавіатуру, див. inputmode
list="…" ID пов'язаного datalist
maxlength="…" Максимальна кількість символів
minlength="…" Мінімальна кількість символів
min="…" Мінімальне числове значення для range & number
max="…" Максимальне числове значення для range & number
step="…" Як число буде збільшуватися в range & number
pattern="…" Вказує регулярний вираз, див. pattern
autofocus Бути у фокусі
spellcheck Виконувати перевірку орфографії
multiple Чи дозволити multiple значення
accept="" Очікуваний тип файлу в елементах керування file завантаження

#Мета-теги

Мета-тег описує метадані в HTML-документі. Він пояснює додаткову інформацію про HTML.

<meta charset="utf-8">

<!-- заголовок -->
<title>···</title>
<meta property="og:title"  content="···">
<meta name="twitter:title" content="···">

<!-- url -->
<link rel="canonical"       href="https://···">
<meta property="og:url"  content="https://···">
<meta name="twitter:url" content="https://···">

<!-- опис -->
<meta name="description"         content="···">
<meta property="og:description"  content="···">
<meta name="twitter:description" content="···">

<!-- зображення -->
<meta property="og:image"  content="https://···">
<meta name="twitter:image" content="https://···">

<!-- ua -->
<meta http-equiv="X-UA-Compatible" content="IE=edge,chrome=1">

<!-- viewport -->
<meta name="viewport" content="width=device-width">
<meta name="viewport" content="width=1024">
